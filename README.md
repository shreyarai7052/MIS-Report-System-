# 📊 Excel MIS Dashboard

An interactive **Management Information System (MIS) Dashboard** built in **Microsoft Excel** to analyze employee performance and operational metrics. The dashboard provides dynamic reports and visualizations that help monitor key performance indicators (KPIs) and support data-driven decision-making.


## 📌 Project Overview

This project showcases the use of Microsoft Excel to create a professional MIS dashboard for business reporting. It transforms raw data into meaningful insights using Pivot Tables, Pivot Charts, slicers, and Excel formulas.

The dashboard focuses on:
- Employee Working Hours Analysis
- Active Customer Performance
- Interactive KPI Reporting
- Business Performance Monitoring


## 🛠️ Tools & Features

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas & Functions
- Data Validation
- Interactive Dashboard Design


## 📊 Dashboard Reports

### 📈 Report 1: Employee Working Hours
- Displays total working hours of each employee.
- Helps compare employee productivity.
- Visualized using a dynamic column chart.

### 👥 Report 2: Active Customers
- Shows active customer contribution by each employee.
- Enables quick performance comparison.
- Visualized using a pie chart.

## 📈 Key Insights

- Compare employee working hours efficiently.
- Track active customer engagement.
- Identify high-performing employees.
- Monitor business KPIs using dynamic reports.
- Improve business reporting through interactive visualizations.

## ▶️ How to Use

1. Clone this repository or download the Excel file.
2. Open the workbook using Microsoft Excel (2016 or later recommended).
3. Enable editing if prompted.
4. Use the filters and slicers to interact with the dashboard.
5. Explore employee performance and customer activity reports.

## 🎯 Project Objective

To design an interactive Excel-based MIS dashboard that enables efficient monitoring of employee performance and customer activity, helping users derive actionable business insights from operational data.
